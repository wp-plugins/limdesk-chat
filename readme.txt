== Description ==

The Limdesk Chat integration plugin for Wordpress.
Add a Limdesk Chat to your Wordpress site.

== Installation ==

1. Upload `limdesk-chat.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Save 'Hash value' in the 'Limdesk Chat' menu in WordPress
1. Profit!

== Frequently Asked Questions ==

= Where I can get hash value =

Sign in to Limdesk account and click Settings -> Chat -> API and Widget
